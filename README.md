# 📚 Book Price & Rating Analysis

A Python web scraping project that extracts book information from the Books to Scrape website and performs basic data analysis and visualization.

## 📌 Features

- Scrape Book Names
- Scrape Book Prices
- Scrape Ratings
- Scrape Availability
- Export Data to CSV
- Price Distribution Graph
- Rating Count Graph

---

## 🛠 Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib
- Seaborn

---

## ▶️ Installation

Clone the repository.

```bash
git clone <repository-url>
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the project.

```bash
python main.py
```
